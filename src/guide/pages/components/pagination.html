Pagination

Pagination은 콘텐츠 또는 데이터를 여러 페이지로 분할하고 다음 또는 이전 페이지로 이동하는 컨트롤입니다.
제공되는 정보의 양과 사용자의 현재 시점을 알려줌으로써 사용자가 혼란 없이 정보를 원활하게 이용할 수 있도록 구성합니다.

기본 가이드

  • nav, a 태그를 사용하여 구조를 만듭니다.

접근성 가이드

  • aria-label="페이지네이션"
  • 페이지 이동이 불필요한 경우 이동 a태그를 없애거나, 비활성화 (tabindex="-1")합니다.
  • 현재 페이지에는 span 태그로 변경, aria-current="page" 를 추가합니다.
  • .hide-txt 로 페이지 이동 기능을 기재해줍니다.
