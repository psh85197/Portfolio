Components 작성법

Components를 작성하는 요령

1. sample.html 복사

  • sample.html 파일을 복사하여 필요한 컨포넌트 파일명으로 변경하여 사용합니다.
  • sample.html을 복사하여 필요한 컨포넌트를 생성하였으면 component_guide.json 파일로 들어가서 생성한 파일명을 추가 합니다.

2. 컨포넌트 파일

  • 컨포넌트파일.html을 열어 컨포넌트를 작성합니다.
  • 작서하기에 앞서 uid-main 클래스 옆에 code-guide 클래스가 추가 되었는지 확인합니다.
  • 컨포넌트의 내용은 code-wrap 클래스 요소의 하위 각각의 code-group에 html, css, javascript를 작성합니다.
  • css 내용 작성에 주석이 필요할시 /* */을 사용하여 주석을 작성합니다.
