1 Components 를 작성하는 요령

  • sample.html 복사
  • sample.html 파일을 복사하여 필요한 컴포넌트 파일명으로 변경하여 사용합니다.
  • sample.html을 복사하여 필요한 컴포넌트를 생성하였으면 component_guide.json 파일로 들어가서 생성한 파일명을 추가 합니다.

2. 컴포넌트 파일

  • 컴포넌트파일.html을 열어 컴포넌트를 작성합니다.
  • 작성하기에 앞서 uid-main 클래스 옆에 code-guide 클래스가 추가 되었는지 확인합니다.
  • 컴포넌트의 내용은 code-wrap 클래스 요소 중 아래 영역에 작성 하면 됩니다.
    <textarea class="code_content html ui_code_content" data-code="html"></textarea>
  • 아래 주석처리 된 영역을 복사하여 사용 합니다.
    <!-- 컴포넌트 작성 단위 -->
    <!-- //컴포넌트 작성 단위 -->
  • iframe 영역에 높이를 줘야할경우
    <textarea class="code_content html ui_code_content" data-code="html"></textarea> 에 data-height="200"을 적용합니다.
