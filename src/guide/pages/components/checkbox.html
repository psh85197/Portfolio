Checkbox

체크박스는 사용자가 선택할 수 있는 옵션을 제공하는 입력 요소입니다.
체크박스는 선택적인 항목을 나타내므로, 사용자는 선택적으로 해당 옵션을 활성화하거나 비활성화할 수 있습니다.

기본 가이드

  • Checkbox 컴포넌트는 input.js의 동작을 위해 반드시 component-input 클래스를 사용합니다.
    이 클래스가 없다면 스크립트가 적용되지 않습니다.
  • 추가적인 스타일의 Checkbox가 필요할 경우, component-input 내부에 -inner 접미사를 가진 클래스를 적용하여 스타일을 확장해야 합니다.
    예를 들어, Checkbox는 checkbox-inner, 스위치 형태의 checkbox는 switch-inner 클래스를 사용하여 스타일을 정의합니다.

협의 할 점

  • 협의점을 작성합니다.

접근성 가이드

  • checkbox - :focus-visible{outline: -webkit-focus-ring-color auto 1px;}
  • switch - :focus-within .switch-item {outline: -webkit-focus-ring-color auto 1px;}
  • switch - aria-label="스위치 제목" 기재

Checkbox

기본 Checkbox 컴포넌트입니다.
  • 구성 요소
    컴포넌트 클래스 : component-input
    하위 요소 클래스 : checkbox-inner

Checkbox + 텍스트

  • 구성 요소
    컴포넌트 클래스 : component-input
    하위 요소 클래스 : checkbox-inner
    텍스트 클래스 : checkbox-txt

Checkbox Switch

  • 구성 요소
    컴포넌트 클래스 : component-input
    하위 요소 클래스 : swich-inner

Checkbox - Agreement

전체 동의 컴포넌트는 component-agreement 클래스를 사용합니다.
data-props-all-check="true" 속성을 적용하면 전체 동의하기 체크박스 선택 시 하위 체크박스들이 자동으로 선택됩니다.
data-props-sub-check="true" 속성을 적용하면 하위 체크박스 중 하나를 선택 시 해당 체크박스에 연관된 서브 체크박스가 선택되는 구조입니다.
  • 구성 요소
    컴포넌트 클래스 : component-input
    하위 요소 클래스 : swich-inner
