Accordion Multiple

아코디언 컴포넌트는 component-accordion 클래스를 사용합니다.
기본값은 type="multiple" 이며, 각 accordion-item 이 개별적으로 열립니다.
특정 accordion-item 을 기본적으로 열어두려면 해당 요소에 data-state="open" 속성을 추가하면 됩니다.

Accordion Single

data-props-type="single" 속성을 적용하면 accordion-item 이 열릴 때 기존에 열린 항목은 자동으로 닫힙니다.
특정 accordion-item 을 기본적으로 열어두려면 해당 요소에 data-state="open" 속성을 추가하면 됩니다.
