Input

기본 Input 컴포넌트입니다.
data-props-clear="true" 속성을 적용하면 입력 필드에 클리어 버튼이 표시됩니다.
  • 구성 요소
    컴포넌트 클래스 : component-input

Input 상태

disabled 또는 readonly 속성을 적용하면 component-input 요소에 input-disabled 및 input-readonly 클래스가 자동으로 추가됩니다.
이 클래스들은 스크립트에서 해당 속성이 설정될 때 동적으로 적용됩니다.
  • 구성 요소
    컴포넌트 클래스 : component-input
    상태 클래스 : input-disabled input-readonly

Input + button

입력 영역과 우측 버튼 영역으로 구성됩니다.
우측 버튼은 공통적으로 input-field-btn 클래스를 사용해야 하며, 해당 클래스는 모든 버튼에 일관되게 적용됩니다.
  • 구성 요소
    컴포넌트 클래스 : .component-input
    공통 클래스 : .input-field-btn

Input file

type="file" 속성을 사용합니다.
버튼 요소는 반드시 input-file-btn 클래스를 적용해야 합니다.
  • 구성 요소
    컴포넌트 클래스 : component-input
    공통 클래스 : input-file-btn

Input + Icon

입력 영역과 우측 아이콘 영역으로 구성됩니다.
우측 아이콘은 공통적으로 input-field-ico 클래스를 사용해야 하며, 해당 클래스는 모든 아이콘에 일관되게 적용됩니다.
  • 구성 요소
    컴포넌트 클래스 : component-input
    공통 클래스 : input-field-ico

Input + Info Text

입력 필드 아래에는 안내 텍스트를 표시하는 info 영역으로 구성됩니다.
input-info 클래스를 적용해야 합니다.
  • 구성 요소
    컴포넌트 클래스 : component-input
    필수 클래스 : input-info

Input + Text

입력 필드의 좌측 또는 우측에 텍스트가 필요할 경우 사용합니다.
input-field-txt 클래스를 적용해야 합니다.
  • 구성 요소
    기본 클래스 component-input
    공통 클래스 input-field-txt

Input 유효성 검사

유효성 검사는 input-valid input-invalid input-reauired 클래스를 통해 상태를 표시합니다.
클래스는 각각 유효한 입력(성공), 잘못된 입력(오류), 필수 입력 상태를 나타냅니다.
  • 구성 요소
    기본 클래스 component-input
    유효성 검사 클래스 : input-valid input-invalid input-required
