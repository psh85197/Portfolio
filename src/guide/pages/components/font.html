Typography 설정

변수에 제목 및 본문텍스트에 사용되는 폰트 스타일을 정의합니다.

Font

  • 디자인 Figma 설정된 값으로 적용한다.
  • 폰트 추가하는 방법
    - font-size line-height font-weight 입력 후 '폰트 추가하기' 버튼 클릭
    - 테이블 추가 및 font mixin 추가됩니다.
  • 추가 한 폰트값을 삭제하는 방법(콘솔창에 입력)
    - 전체 폰트 삭제 : window.clearFontsFromTable()
    - 특정 폰트 크기만 삭제 : window.clearFontsFromTable('25')
  • 위 내용 참고 후 컴포넌트 관리도구를 통해 가이드 추가 - README-COLOR-MANAGER.md 파일 참고
